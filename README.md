## What is DSA Visualizer?

DSA Visualizer is a full-stack web application that makes learning algorithms easier and more engaging. Instead of reading dry textbook definitions, users can:

- Browse a curated library of algorithms across 8+ categories
- View clean implementations in **Python**, **JavaScript**, and **C++**
- Understand **time & space complexity** with color-coded Big-O notation
- Launch **interactive visualizations** on VisuAlgo with one click
- Search and filter by category and difficulty level

---

## Features

| Feature | Description |
|---------|-------------|
| 🔍 Smart Search | Debounced real-time search across names, tags, and descriptions |
| 📂 Category Filter | Filter by Sorting, Graph, DP, Tree, Greedy, and more |
| 🎯 Difficulty Filter | Easy / Medium / Hard difficulty levels |
| 💻 Multi-language Code | Python, JavaScript, C++ with syntax highlighting |
| 📋 Copy to Clipboard | One-click code copying |
| ⚡ Complexity Analysis | Best, average, and worst case Big-O breakdowns |
| 🎬 Visualizations | Direct links to VisuAlgo and YouTube explanations |
| ❤️ Like System | Like your favourite algorithms |
| 📱 Responsive | Works on mobile, tablet, and desktop |
